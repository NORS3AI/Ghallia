# Ghallia

**A medieval idle/active crafting game inspired by RuneScape and World of Warcraft professions.**

[![Play Now](https://img.shields.io/badge/Play%20Now-Ghallia-gold?style=for-the-badge&logo=gamepad&logoColor=white)](https://nors3ai.github.io/Ghallia/)

---

## About

Ghallia is a hybrid idle/active browser game where you:

- **Gather resources** by actively tapping (Logging, Mining, Fishing...)
- **Craft items** passively in idle queues (Sawmill, Smithing, Cooking...)
- **Sell goods** for gold to unlock more skills
- **Prestige** to earn Chaos Points and permanent upgrades
- **Progress** through 999 levels per skill with 50 unique tiers each

## Features

- **20 Skills** - 7 Gathering, 9 Crafting, 4 Support (post-prestige)
- **Levels 1-999** - Deep progression with tier unlocks every 20 levels
- **Prestige System** - Reset for Chaos Points, unlock talent trees
- **Mobile-First** - Optimized for iPhone with swipe navigation
- **Medieval Theme** - Dark aesthetic with gold accents
- **Offline Progress** - Crafting continues while you're away
